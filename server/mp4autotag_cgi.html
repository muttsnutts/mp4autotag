mp4autotag_cgi

mp4autotag_cgi is a remote procedure for MP4Autotag.

With a given file name string it will determine if the file is a show or a movie and then will return a json representation of a mp4_tag array of search results from themoviedb.org and thetvdb.com. The api follows the RESTFul standard.

Usage:

  • http://popmedic.com/cgi/mp4autotag_cgi.rb/search=<filename>&use_itunes=<0:1>

    Where

    <filename> is the name of a file to parse and get movies or shows from.
    <0:1> is optional, but if set to >= 1 will return results with itunes images.
